# Examen 2º SMR: 1ª Evaluacion (B) 2024

## Instrucciones
1. Clona este repositorio con Github Desktop
2. Completa el código de la web siguiendo los pasos del PDF que hay en la tarea de TEAMS
3. Haz COMMIT de vez en cuando para guardar tu progreso y escribe en la descripción "TuNombre HH:MM", donde TuNombre es tu nombre y HH:MM es la hora a la que haces el COMMIT
4. NO hagas PUSH hasta que hayas terminado
5. Cuando hayas terminado haz un último COMMIT y un PUSH para entregar el examen
6. Escribe tu nombre de usuario de Github en la tarea y entregala
